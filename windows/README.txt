== XP und Vista ==
Die Treiber die hier zu finden sind, wurden auf XP-Systemen geschrieben und
getestet. Es gibt aber auch vereinzelte Berichte über Erfolge auf
Vista-Systemen. Bitte auf der Mailingliste über Erfolge (aber auch Probleme)
berichten, dann könnte das Verzeichnis „XP“ in „XP_und_Vista“ umbenannt werden.
