  • Navigation Keys also lie directly under your finger tips! ←↑→↓ (on Layer 4)

Installation

Due to its rich layers, programs can behave strange (on upper layers). We know, that they are just overwhelmed ;)

ABSOLUTELY NO WARRANTIES FOR ANYTHING!
USE AT YOUR OWN RISK!

Windows

You'd also want to have a font like Deja Vu or Libertine to see all the fancy Neo–typeable chars!

  • Try–out and Non–Administrators: Just run neo20.exe. To deactivate and switch back: Double-click on the Neo–Trayicon, or use [Shift]+[Pause]! (Peek at Neo: [CapsLock]+[F1])
  • Administrator Priviledges: After System backup, look at kbdneo32.zip for a 32–, or at kbdneo64.zip for a 64–Bit Windows: Run reg_backup.bat (and get your registry keys for deinstalling Neo!), copy kbdneo2.dll to C:\Windows\system32\ per Windows Explorer(!), run kbdneo2_install.reg. Now there's Neo under the Control panel's keyboard configuration task as a variant after selecting German as keyboard language. For using cursors etc. you will also have to run kbdneo_ahk.exe. To deactivate and switch back: Use the keyboard layout icon, or press [Ctrl]+[Shift]. Deinstallation: Either only choose your old layout, or purge Neo away: After deselecting Neo as keyboard layout, run kbdneo2_uninstall.reg, and run the registry keys got at installation by reg_backup.bat

(Layers 4 and 6 are not accessible right now :|)Get neo.keylayout (if necessary, rename it to this name!) and neo.icns, move both in your directory at ~/Library/Keyboard Layouts, log out and in again, now you can select Neo at the normal Keyboard configuration panel as layout. Mod3 is at ⌥.
