== Einsatz ==
Dieser Flyer ist im A4-Format, damit ihn jeder ohne Probleme ausdrucken und kopieren kann – gerade deshalb gibt es auch eine reine Schwarz-Weiß-Version.

== Kompilieren ==
Eine installierte LaTeX-Distribution vorausgesetzt, kann mit
      make
eine pdf-Datei erzeugt werden.

== Geschichte ==
Der Flyer wurde schon mehrfach überarbeitet:
• Die erste Entwurf wurde von Frank Stähr mit Scribus erstellt,
• dann konvertierte Stefan Mayer ihn in pdfLaTeX,
• diese Fassung wurde dann wieder von Frank Stähr überarbeitet und erweitert.
